new.spirit Documentation

Actually there is no new.spirit documentation available.

The CIPP documentation is contained in Perl Module
CIPP::Manual. Use the command 

  perldoc CIPP::Manual

to read the documentation. You'll find a (probably slightly older)
PDF version of the CIPP documenation on CPAN:

http://www.perl.com/CPAN/modules/by-module/CIPP/

Download the package CIPP-Doc-x.xx.tar.gz, it contains the PDF file.
